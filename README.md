# DevOps Lab 2 – Zadanie 2

To repozytorium przedstawia prosty projekt w Pythonie używany do demonstracji **Continuous Integration (CI)** z wykorzystaniem **GitHub Actions** oraz powiadomień wysyłanych do Discorda przez webhook.

---

## Zawartość projektu

- Aplikacja Python: `src/app.py`  
- Test jednostkowy: `tests/test_app.py`  
- Workflow CI: `.github/workflows/ci.yml`  
- Powiadomienia o błędach przez Discord Webhook
